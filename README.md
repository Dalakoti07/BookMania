# BookReview
Adapted and solution to `first project` of Harvard cs50 web development course `https://www.edx.org/course/cs50s-web-programming-with-python-and-javascript`

## How To Open
- create a python virtual env and activate it `https://uoa-eresearch.github.io/eresearch-cookbook/recipe/2014/11/26/python-virtual-env/`
- `pip install -r requirements.txt`
- Open the command promp and type `python application.py`
